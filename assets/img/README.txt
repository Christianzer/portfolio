==========================================================================
  CAPTURES D'ÉCRAN DES PROJETS  —  comment les ajouter
==========================================================================

Chaque carte de projet affiche par défaut un APERÇU GÉNÉRÉ (mockup SVG).
Pour le remplacer par une VRAIE capture d'écran, déposez simplement une
image dans CE dossier, nommée avec l'identifiant du projet.

  Format accepté : .png  (ou .jpg)
  Format conseillé : 16:9  (ex. 1280 x 720 px)
  Nom du fichier   : <id>.png

L'image s'affiche automatiquement à la place du mockup, sans rien coder.
Si aucune image n'est trouvée, le mockup généré reste affiché.

--------------------------------------------------------------------------
 IDENTIFIANTS (id) À UTILISER POUR LES NOMS DE FICHIERS
--------------------------------------------------------------------------
  khadim.png        -> Khadim (Master) — ERP
  fne.png           -> FNE — Facture Normalisée
  nalacare.png      -> NalaCare
  dubass.png        -> DUBASS
  beautelic.png     -> Beautelic
  mmm.png           -> Mobile Money Master
  qiryna.png        -> Qiryna
  kolivri.png       -> Kolivri
  classeia.png      -> Classe IA
  eimmo.png         -> e-immo
  gesope.png        -> GESOPE
  ciat.png          -> ciatci.com + BI
  multimarket.png   -> MultiMarket
  modelis.png       -> Modelis Tech

Exemple : pour mettre la vraie capture de FNE,
placez le fichier  ->  assets/img/fne.png
